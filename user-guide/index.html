User Guide

This section provides a comprehensive guide to using EOS in your own research. In the first subsection, the basic concepts, building blocks, and terminology of EOS are introduced. The next three subsections explain how to use EOS’ low-level interface to perform the three main tasks of EOS: how to make theoretical prediction; how to infer parameters from experimental and/or theoretical data; and how to simulate processes.
